# Python Project Setup for ManimGL

This directory contains Python scripts for creating mathematical animations using ManimGL.

## Setup

1. Create a virtual environment (recommended):
```bash
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the example animation:
```bash
manimgl examples/basic_example.py BasicScene
```

For more information about ManimGL, visit: https://github.com/3b1b/manim
