# Side project for Centrale Lyon
### Lightweight menu bar application made to automate the entering of the password in the school wifi board 


Username/Passwords saved locally on your machine , no connection whatsoever to a server for security reasons.
Made with tauri (rust backend, js/html frontend)

### Setup:
Download ecl+.dmg
give the app accessibility permission (for autofill)
submit your credentials and press CMD/Windows + R on the ecl wifi page to autoconnect
